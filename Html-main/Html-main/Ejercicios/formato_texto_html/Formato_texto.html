HTML (HyperText Markup Language) es el lenguaje utilizado para crear y diseñar páginas web en la World Wide Web. Fue creado en 1989 por Tim Berners-Lee, un científico de la computación británico que trabajaba en el Centro de Investigación del CERN en Suiza.

Berners-Lee estaba buscando una manera de compartir información entre científicos de todo el mundo, y creó un sistema que permitía a los usuarios acceder a documentos en línea mediante hipervínculos. Para crear y diseñar estos documentos, Berners-Lee creó un lenguaje de marcado simple que llamó HTML.

La primera versión de HTML fue muy básica, pero permitió a los usuarios crear documentos que podían ser enlazados entre sí. Con el tiempo, se agregaron más etiquetas y características a HTML, lo que permitió a los diseñadores crear páginas web más sofisticadas y complejas.

A medida que la World Wide Web crecía en popularidad, HTML se convirtió en el lenguaje estándar para crear páginas web. En 1995 se publicó la versión 2.0 de HTML, que introdujo nuevas etiquetas y características, y en 1997 se publicó la versión 3.2 de HTML, que se convirtió en la versión estándar de facto.

En los años siguientes, se publicaron varias versiones más de HTML, incluyendo HTML 4.01 en 1999 y XHTML (eXtensible HTML) en 2000. XHTML fue una versión reformulada de HTML que se basó en XML (eXtensible Markup Language) y se centró en la semántica y la accesibilidad.

En 2010, se publicó la versión 5 de HTML, que se convirtió en el estándar actual para crear páginas web. HTML5 incluyó muchas nuevas etiquetas y características que permitieron a los diseñadores crear páginas web más ricas y dinámicas, con efectos visuales, audio y video integrados.
