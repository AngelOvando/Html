Elementos basicos

Las etiquetas <h1> a <h6> se utilizan para crear encabezados o títulos en diferentes niveles de importancia.

Las etiquetas <ul> y <li> se utilizan para crear listas sin ordenar.

Las etiquetas <ol> y <li> se utilizan para crear listas ordenadas.

Las etiquetas <table>, <thead>, <tbody>, <tr>, <th> y <td> se utilizan para crear tablas y definir sus encabezados, filas y celdas.

Las etiquetas <form>, <input>, <label>, <select>, <option>, <textarea> y <button> se utilizan para crear formularios y definir sus campos, etiquetas y botones de envío.

La etiqueta <br> se utiliza para insertar un salto de línea dentro de un texto.

La etiqueta <hr> se utiliza para crear una línea horizontal que separa secciones en una página web.

La etiqueta <meta> se utiliza para incluir metadatos, como la descripción y las palabras clave, en la cabeza de un documento HTML para fines de SEO y accesibilidad.
